整形外科リウマチ科HPブラッシュアップ

長年運用されている整形外科・リウマチ科クリニックのウェブサイトを、理事長様のご希望に沿ってブラッシュアップしました。
既存の雰囲気を崩さずに、増加した診療科目で複雑化していたヘッダー・ナビゲーションを整理し、直感的で使いやすい導線設計へ改善。情報量を保ちながらも、すっきりとした印象に仕上げました。

また、旧FC2ドメインから新サーバーへの移行を実施し、安定した運用環境を構築。さらにGoogle広告運用も担当し、集患につながるWebマーケティング施策を強化しています。

■ 担当範囲

・ バナー・グラフィック素材作成
・ コピーライティングおよびコンテンツ構成
・ Webデザイン・フロントエンドコーディング（Illustrator、Photoshop、Dreamweaver使用）
・ レスポンシブ対応（PC・タブレット・スマホ最適化）
・ ドメイン取得・DNS管理・リダイレクト設定
・ SEO対策およびGoogleビジネスプロフィール統合
・ 運用・更新業務

■ 使用ツール / 技術

・ Illustrator
・ Photoshop
・ Dreamweaver
・ GA4 / GSC
・ HTML / CSS / JavaScript
